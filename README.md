# Quiz-application-
This script will display questions one by one, ask the user for their answer, and calculate a final score at the end
1️⃣Separation of Data and Logic: By keeping quiz_data separate from the run_quiz function, you can change the questions without touching the code logic.
​2️⃣strip().upper(): This makes the quiz user-friendly by ignoring accidental spaces and accepting both lowercase ("a") and uppercase ("A") inputs.
3️⃣​Scalability: You can easily add more questions to the quiz_data list without needing to change any of the code.
